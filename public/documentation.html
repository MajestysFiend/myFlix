myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies!

Request URL Response
Get a list of all movies /movies A JSON object containing data on all movies
Get data about a single movie by title /movies/:title A JSON object containing data on a single movie title
Get data about a genre on a single movie title /movies/:title/genre A JSON object containing data on a single movies genre
Get data about a director by name /movies/:director A JSON object containing data on a single director
New user registration /user A text message indicating a successful/unsuccessful registration
Update user information /user A text message indicating user information was successfully/unsuccessfully updated
Add a movie to your list of favorites /user/favorites A text message indicating a movie was successfully/unsuccessfully added to favorites
Remove a movie from your list of favorites /user/favorites A text message indicating a movie was successfully/unsuccessfully removed from favorites
User deregisteration /user A text message indicating the user was successfully/unsuccessfully removed
